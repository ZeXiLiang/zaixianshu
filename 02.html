配置GitBook

配置GitBook

安装gitbook 在开始-附件-命令提示符-右键用管理员运行 打开输入

npm install gitbook-cli -g
找一个盘。 比如：在我的E盘中 新建文件夹-重命名为【02】文件夹 。 开始-附件-命令提示符-直接打开 不用管理员运行 进入E盘的02文件夹下。 命令是：两个步骤输入【e:】和【cd 02】。如图 .gitbook初始化 输入命令

gitbook init
预览命令 
gitbook serve
在浏览器地址栏输入 http://localhost:4000 可以预览你编写的md文件。 生成HTML文件。 
gitbook build
会在书籍的文件夹中生成一个 _book 文件夹，这个文件夹就是生成的html文件

构建目录

此时该文件夹下会生成俩文件【README.md】和【SUMMARY.md】 README.md: 书的介绍文字，如前言、简介，在章节中也可做为章节的简介。 SUMMARY.md: 定制书籍的章节结构和顺序。 目录的语法如下：* [显示的目录名](文件夹/文件名#标题id)。 下一级：- [显示的目录名](文件夹/文件名#标题id)

MarkDown语法

在根目录空白处-右键新建文本文档-后缀改为【01.md】。用下载好的MarkDownPad编辑器编写文档。 参考简易MarkDown语法详解 你可以在左边实时编写，右边即可出现效果。

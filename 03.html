GitHub展示

GitHub展示

在github新建账号

登陆到默认页面 点击右侧+号 New repository

填写项目名称

打对勾

最后点击Creact repository

下载git 选择最新版本中的 仓库名下载 例如：Git-2.21.0-64-bit.exe下载 选择相应系统64或32位

安装根据默认一直点击下一步 点8，9个默认下一步

环境变量配置

右键-计算机-属性-高级属性设置-高级-环境变量-系统变量名称【PATH】：值：【C:\Program Files\Git\cmd】多个值之间只需用小写【;】分割

前提条件1：找到你在github上面的账号名称：例如我的名称是ZeXiLiang

前提条件2：在头像下拉列表中找到Settings 进入设置页面

左侧选择Emails 右侧设置邮箱账号名称 打上两个对勾 取得加密邮箱字符串 例如我的是： 16136559+ZeXiLiang@users.noreply.github.com

开始-附件-命令提示符-右键用管理员身份运行 输入 git 再输入 git-version 再在命令行输入 

git config --global user.name "ZeXiLiang"
git config --global user.email " 16136559+ZeXiLiang@users.noreply.github.com"

查看Git配置

查看当前用户名

git config --global user.name
# 设置用户名 
git config --global user.name "ZeXiLiang"
# 查看当前电子邮件地址 
git config --global user.email
# 设置电子邮件地址 
git config --global user.email "16136559+ZeXiLiang@users.noreply.github.com"

新建本地仓库与网上仓库链接

选择E盘 右键空白处点击Git Bash here(git完成安装后就会出现)

新建文件夹并重命名03 进入03 【注意流程 从大目录到小目录】

进入网上仓库 如 我的仓库 这个网址加git 命令参考：git clone + 网上仓库+.git

git clone https://github.com/ZeXiLiang/zaixianshu.git

这时候E盘03文件夹多出个网上仓库文件夹【zaixianshu】里面有.md文件 把章节2中本地生成的静态网页文件夹【_book】 拖入在本地多出的【在线书】文件夹中

再cd 进入多出的网上仓库文件夹【zaixianshu】

输入命令 查看仓库状态

git status

输入命令 添加当前目录下的所有文件和子目录

git add .

再输入看到变绿色 表示它们准备好了提交

git status
![denglu](http://wp.yunxiu520.com/userfiles/01/tupian/20.jpg) 网上仓库 点击Settings （跟头像那个设置位置不同） ![denglu](http://wp.yunxiu520.com/userfiles/01/tupian/17.jpg) 鼠标滚轮滑动到底部找到GitHub Page区块，点击Source中的none切换为master branch -点击Save ![denglu](http://wp.yunxiu520.com/userfiles/01/tupian/19.jpg) 点击Save后出现一个网址 点击进入网上仓库搭建的展示在线书籍 页面 输入命令 上传至远程master分支（也就是为本地的文件夹上传的网上仓库做准备） 
git commit -m "modify"

输入命令 

git pull 

git push
输入最后一行git push回车 等会 会有弹出框让你输入GitHub的账号密码 嗯我没输入过 有的话你就输入 没有弹出框就刷新网上仓库
